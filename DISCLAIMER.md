## 🔒 Technical Experiment

This project implements advanced smart contract patterns including:
- EIP-5114 soulbound tokens
- UUPS proxy upgradability with timelock
- ERC-1155 ephemeral access tokens
- EIP-712 signature-based metadata updates

The code serves as a technical demonstration of modular credential infrastructure for Web3 identity systems.
